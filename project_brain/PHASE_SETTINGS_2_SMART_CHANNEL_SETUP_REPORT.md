## UI Behavior

1. **Step 1** — User enters “What is your channel about?” with example hints.
2. **Step 2** — Clicks **Generate Channel Profile** (calls suggest API).
3. **Step 3** — Preview banner: *Generated by AI — please review before saving*; all fields editable with dropdown presets (language, tone, visual style, platform, duration, provider, narration, music, upload platforms, topic lists).
4. **Step 4** — User clicks **Save Channel Profile** (existing `POST /product/channel-profile`).

Copy shown: *Custom Topic in Create Video will not change this saved profile.*

## API Behavior

### `POST /product/channel-profile/suggest`

**Input:**
```json
{
  "channel_topic": "Scorpions and desert animals",
  "language_preference": "",
  "platform_preference": ""
}
```

**Output:** `ChannelProfileSuggestionDTO` (structured suggestion + `reasoning` + `source`).

**Does not write** `project_brain/product_settings/channel_profile.json`.

### `POST /product/channel-profile`

Unchanged — persists after user confirmation.

## Create Video Integration

On load, Create Video reads saved profile:

- `default_provider`, `default_platform`, `default_duration_seconds`
- `visual_style` (falls back to `tone_style`) for the style field
- `use_ai_director_default` → AI Director ON
- `use_prompt_critic_default` → Prompt Critic ON

Custom Topic mode overrides only the video topic via preflight; channel profile JSON is unchanged.
